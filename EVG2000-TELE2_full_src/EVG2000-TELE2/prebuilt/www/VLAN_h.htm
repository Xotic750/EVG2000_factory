VLAN Group Settings

Port Based
Differentiate VLAN by port.

To edit the port mapping configuration:

1.Port Mapping supports multiple ports to WAN groups. Each group will perform as an independent network.

2.To support this feature, you could add interfaces to the grouped list using the "Add" button.

3.The "Remove" button will remove interfaces from group and add its to the Default group.

4.Click "Apply/Reboot" button to make the changes effective immediately

5.Click "Cancel" button to give up the changes.

6.If you make any changes to vlan you will need to restart IAD for the settings to take effect.
