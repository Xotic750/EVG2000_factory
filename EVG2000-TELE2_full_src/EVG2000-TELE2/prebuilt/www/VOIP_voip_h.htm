VoIP Settings Help

This page shows the settings for your VoIP (Voice over Internet Protocol) connection.

Provisioning Service

  • TFTP Provisioning - this option should only be enabled if your unit was pre-configured for your VSP. If you disable this setting you must manually enter all your SIP settings on this page.

SIP Service

  • SIP Proxy - Enter the address of the SIP Proxy.
  • SIP Control Port - Enter the Control Port used by the SIP Server.
  • Starting RTP Port - Enter the beginning of the range of RTP Ports used by the SIP Server.
  • Use Outbound SIP Proxy - Enable this if you want to use the Outbound SIP Proxy function.
  • Outbound SIP Proxy Address - Enter the address of the SIP Proxy used by your VoIP provider.

Line 1/Line 2/Line 3/Line 4

  • Line Enable - Enable this to use VoIP. You need to have an account with a VoIP provider (sometimes called a SIP Server account).
  • Display Name - This name is used by the SIP Server, and may be visible to callers.
  • Telephone Number - Enter the telephone number provided to you by the VoIP Service provider. If you have a range of numbers, enter the number you wish to use for this line.
  • User Name - Enter the login name for connections to the SIP Server.
  • Password - Enter the password associated with the login name above.
  • Register Expiry Time - This set the "Idle Timeout" for the SIP Server Login. An Idle connection will be terminated after this time period. Enter the desired value.

To Save Or Cancel Changes

Click Apply to have your changes take effect.
Click Cancel to return to the previous unsaved settings.
