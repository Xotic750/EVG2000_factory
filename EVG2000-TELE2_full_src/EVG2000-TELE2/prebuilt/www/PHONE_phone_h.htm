Phone Mapping Setting Help

This page allows user to configure VoIP Account to Endpint Phone mapping.

Outgoing Call Mapping

  • Support Many to One Mapping: Different Endpoint Phones can use the same VoIP Account to go out. DECT 1~5 indicate DECT HandSet ID.

Incoming Call Mapping

  • Support Many to Many Mapping: When there is an incoming call, all Endpoint Phones mapped to this VoIP Account will ring and can accept call. DECT 1~5 indicate DECT HandSet ID.

Extension Table

  • Internal Call Extension: Set internal call extension for each Endpoint Phone. Extension should be 3 characters, the first two characters should be digits and the last one must be '*'.

To Save Or Cancel Changes

Click Apply to have your changes take effect.
Click Cancel to return to the previous unsaved settings.
