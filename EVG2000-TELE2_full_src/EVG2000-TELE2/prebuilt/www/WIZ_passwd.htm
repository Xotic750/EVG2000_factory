Change the Administrator Password

    Your NETGEAR router uses an administrator password. It is printed on the bottom label of the router. The default password is "password". The password is case sentive. For example, PASSWORD is not the same as passWORD.

    You can change the password to restrict access to the configuration options of the router.

    Enter a new password or click Next to keep the default password.
