T069 Configuration Help

The TR-069 WAN management protocol allows auto-configuration, provisioning, collection and diagnostics to be performed by an Auto-Configuration Server (ACS). TR-069 Client settings can be input here.

Inform Status

Here you can select whether the EVG2000 can send data to the Auto-Configuration Server.

Inform Interval

The EVG2000 will send inform data to ACS at the assigned time interval periodically in order to get updated information .

ACS URL

Input the IP address of the ACS server that the EVG2000 will connect to.

ACS Username

Enter the username the EVG2000 will use to connect to the ACS server.

ACS Password

Enter the password the EVG2000 will use to connect to the ACS server.
