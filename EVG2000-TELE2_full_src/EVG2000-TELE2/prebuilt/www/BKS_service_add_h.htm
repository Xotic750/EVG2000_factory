Service Definition Help

The Service Definition menu allows you to define or edit your own custom service definition. The custom service can then be used when creating firewall rules.

Name

Enter a suitable name for this Service.

Type

Select the correct type for this Service (if in doubt, select TCP/UDP):

  • TCP
  • UDP
  • TCP/UDP

Start Port

  • If a port range is required, enter the beginning of the range here, and the end of the range in the "Finish Port" field.
  • If a single port is required, enter the same value in both the "Start" and "Finish" fields.

Finish Port

  • This field can be used to enter the end of the range of port numbers.
  • If a single port is required, enter the same value in both the "Start" and "Finish" fields.
