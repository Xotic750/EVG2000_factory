Set Password Help

You use the Set Password page to change the password you use to access the Settings pages -- NOT your ISP account password.

To change the password:

  1. Type the old password.
    The default password for the router is: password.
  2. Type a new password and type it again in the Repeat New Password box to confirm it.
    Note: Be sure to write it down in a secure location.
  3. Click Apply to have the password changed or click Cancel to keep the current password.

Note: If you have backed up the router settings previously, you should do a new backup so that the settings file includes the new password.
